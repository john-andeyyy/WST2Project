Lesson 1: Introduction to C#

What is C#?

C# (pronounced as "C sharp") is a powerful programming language developed by Microsoft. It's designed to be versatile, allowing developers to create a wide range of applications, including interactive websites, mobile apps, video games, augmented reality (AR), virtual reality (VR), desktop applications, and back-end services.

For instance, popular applications like Pokemon Go and Stack Overflow were built using frameworks that utilize C# (Unity and ASP.NET, respectively).

Run Some C#

Let's start with a simple example. The following line of code prints "Hello World!" to the console:

Console.WriteLine("Hello World!");

Getting Input

You can interact with users by capturing their input. Here's how you can prompt the user for their age and display it back to them:

Console.WriteLine("How old are you?");
string input = Console.ReadLine();
Console.WriteLine($"You are {input} years old!");

Comments

Comments are essential in programming. They provide context, improve readability, and allow developers to ignore specific lines of code. In C#, comments can be single-line (starting with //) or multi-line (enclosed within /* */).

Here's an example of how comments can be used:

// This is a single-line comment
/* This is a multi-line comment */
int persnicketyCount = 0; // Variable declaration with a comment

Object-Oriented Programming (OOP) in C#

C# is an object-oriented programming language, which means it uses objects to model real-world entities. Here are some core concepts of OOP in C#:

Classes and Objects

A class is a blueprint for creating objects. It defines properties and methods that the objects created from the class will have. Here's an example of a simple class and how to create an object from it:

public class Person {
  public string Name { get; set; }
  public int Age { get; set; }
}

// Creating an object of the Person class
Person person = new Person();
person.Name = "John";
person.Age = 30;
Console.WriteLine($"Name: {person.Name}, Age: {person.Age}");

Inheritance

Inheritance allows a class to inherit properties and methods from another class. This helps to reuse code and establish a hierarchy. Here's an example:

public class Animal {
  public void Eat() {
    Console.WriteLine("Eating...");
  }
}

public class Dog : Animal {
  public void Bark() {
    Console.WriteLine("Barking...");
  }
}

Dog dog = new Dog();
dog.Eat(); // Inherited method
dog.Bark(); // Method of Dog class

Encapsulation

Encapsulation is the concept of wrapping data and methods into a single unit (class) and restricting access to some of the object's components. This is done using access modifiers. Here's an example:

public class Account {
  private decimal balance;

  public void Deposit(decimal amount) {
    if (amount > 0) {
      balance += amount;
    }
  }

    public decimal GetBalance() {
    return balance;
  }
}

Account account = new Account();
account.Deposit(100);
Console.WriteLine($"Balance: {account.GetBalance()}");

Polymorphism

Polymorphism allows methods to do different things based on the object it is acting upon, even if the method name is the same. There are two types of polymorphism in C#: compile-time (method overloading) and runtime (method overriding). Here's an example of runtime polymorphism:

public class Animal {
  public virtual void MakeSound() {
    Console.WriteLine("Some generic animal sound");
  }
}

public class Dog : Animal {
  public override void MakeSound() {
    Console.WriteLine("Bark");
  }
}

public class Cat : Animal {
  public override void MakeSound() {
    Console.WriteLine("Meow");
  }
}

Animal myAnimal;
myAnimal = new Dog();
myAnimal.MakeSound(); // Output: Bark

myAnimal = new Cat();
myAnimal.MakeSound(); // Output: Meow

Abstraction

Abstraction is the concept of hiding the complex implementation details and showing only the essential features of an object. This is usually achieved using abstract classes and interfaces. Here's an example using an abstract class:

public abstract class Shape {
  public abstract double GetArea();
}

public class Circle : Shape {
  public double Radius { get; set; }

  public Circle(double radius) {
    Radius = radius;
  }

  public override double GetArea() {
    return Math.PI * Radius * Radius;
  }
}

Circle circle = new Circle(5);
Console.WriteLine($"Area of the circle: {circle.GetArea()}");
